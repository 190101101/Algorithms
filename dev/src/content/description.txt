### Counting Sort Algoritması
Counting Sort, belirli bir aralıktaki tamsayıları sıralamak için kullanılan kararlı (stable) bir sıralama algoritmasıdır. Bu algoritma, sayılar arasında sıklıkla geçen değerlerin sayısını hesaplar ve bu bilgiyi kullanarak elemanları sıralar. Negatif sayılar için genellikle kullanılmaz veya algoritma biraz değiştirilir.

#### Counting Sort Ne İçin Kullanılır?
- Küçük ve sınırlı aralıktaki tamsayıları sıralamak için idealdir.
- Büyük sayılar veya geniş aralıklar için etkili değildir.
- Sıralama sırasında ek bellek kullanır, bu nedenle büyük veri kümeleri için uygun olmayabilir.

### Counting Sort Algoritmasının Adımları
1. Verilen dizideki en büyük değeri bul.
2. Her bir değerin kaç kez geçtiğini saymak için bir sayaç (count) dizisi oluştur.
3. Sayaç dizisini kullanarak sıralı bir çıktı dizisi oluştur.

### Counting Sort'un Koduyla Bir Örnek
Aşağıda Counting Sort algoritmasının JavaScript ile uygulanmış bir örneği bulunmaktadır.

```javascript
// Функция для выполнения Counting Sort
function countingSort(arr) {
    // Находим максимальное значение в массиве
    let max = Math.max(...arr);

    // Создаем массив для подсчета с нулями
    let count = new Array(max + 1).fill(0);
    let output = new Array(arr.length).fill(0);

    // Подсчитываем количество каждого элемента в исходном массиве
    for (let i = 0; i < arr.length; i++) {
        count[arr[i]]++;
    }

    // Изменяем count[i] так, чтобы count[i] содержал фактическую позицию этого числа в output
    for (let i = 1; i <= max; i++) {
        count[i] += count[i - 1];
    }

    // Строим выходной массив
    for (let i = arr.length - 1; i >= 0; i--) {
        output[count[arr[i]] - 1] = arr[i];
        count[arr[i]]--;
    }

    // Копируем отсортированные элементы обратно в оригинальный массив
    for (let i = 0; i < arr.length; i++) {
        arr[i] = output[i];
    }
}

// Пример использования
const arr = [4, 2, 2, 8, 3, 3, 1];
console.log("Original array:", arr);

countingSort(arr);
console.log("Sorted array:", arr);
```

Bu kod, Counting Sort algoritmasını kullanarak verilen bir dizi sayıyı sıralar. İlk önce en büyük sayıyı bulur, sonra her bir sayının kaç kez geçtiğini sayar ve bu bilgiyi kullanarak sıralı bir çıktı dizisi oluşturur.
