### Linear Search Nedir?
Linear Search, sıralı veya sırasız bir veri kümesinde aranan bir değeri bulmak için kullanılan basit bir arama algoritmasıdır. Bu algoritma, dizinin ilk elemanından başlayarak her elemanı kontrol eder ve aranan değeri bulana kadar devam eder.

### Linear Search'ün Kullanım Alanları
Linear Search, küçük veri kümelerinde veya verilerin sıralı olmadığı durumlarda kullanılır. Algoritma basit ve anlaşılır olduğu için bazen daha karmaşık algoritmalardan önce hızlıca kontrol yapmak için de kullanılabilir.

### Linear Search Algoritması ve Örneği
Aşağıda Linear Search algoritmasının JavaScript kodunu ve örnek bir kullanımını bulabilirsin:

```javascript
// Функция для выполнения линейного поиска
function linearSearch(arr, x) {
    // Проходим по каждому элементу массива
    for (let i = 0; i < arr.length; i++) {
        // Если элемент совпадает с искомым значением, возвращаем индекс
        if (arr[i] === x) {
            return i;
        }
    }
    // Если элемент не найден, возвращаем -1
    return -1;
}

// Örnek kullanım
const array = [2, 3, 4, 10, 40];
const x = 10;
const result = linearSearch(array, x);
if (result !== -1) {
    console.log("Element found at index:", result); // Konsolda elemanın bulunduğu indeksi yazdır
} else {
    console.log("Element not found in the array"); // Konsolda elemanın bulunamadığını yazdır
}
```

Bu kodda, Linear Search algoritmasının temel adımları şu şekildedir:
1. `linearSearch` fonksiyonu, diziyi baştan sona dolaşır.
2. Her elemanı kontrol eder ve aranan değere eşit olup olmadığını kontrol eder.
3. Eşit olduğunda, elemanın indeksini döner. Eğer aranan değer bulunamazsa, -1 döner.
